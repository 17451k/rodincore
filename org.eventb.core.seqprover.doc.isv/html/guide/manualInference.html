Manual Inference Reasoners

Concepts

The reasoners of this type does not have a single string input (except the sequent) representing the position in the formula (either a hypothesis or the goal) where most appropriate to indicate which inference rule to apply. For example, a reasoner can implement the following inference rules which dealing with conjunction in the hypothesis and in goal. 

  H, P |- Q
-----------------  
 H |- P => Q
