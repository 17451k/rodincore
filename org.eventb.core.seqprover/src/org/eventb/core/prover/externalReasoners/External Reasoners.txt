
External Reasoners
==================


Currently Implimented

AllF : Instantiating a universally quantified hypothesis
ConjD : Removing conjunction in the hypothesis
Contr : Starting a proof by contradiction
Cut : Inserting a Lemma
DisjE : Starting a case distinction using a disjunctive hypothesis
Eq : Using an equality to replace all uccurances of a free variable
ExF : Removing an existential from the hypothesis, making its variables free
ExI : Providing witnesses for an existential goal
ImpD : Removing an implicative hypothesis
LegacyProvers : Old B provers PP/ML
Trivial : Discharges a sequent with true goal or false hypothesis.
