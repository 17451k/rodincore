Adding a POM-tactic

POM tactics are the tactics that run by the auto POM (Reference ?). Only automatic tactics (See About tactics) can be declared as POM tactic.

POM tactics are added using the extension point org.eventb.core.pomTactic. An example is as follows: 

   <extension
         point="org.eventb.core.pomTactics">
      <tactic id="org.eventb.core.seqprover.trueGoalTac"/>
   </extension>

Each tactic element contains only attribute id which must be a valid tactic ID (including the project prefix) as declared using org.eventb.core.seqprover.autoTactics extension point.

Once declared as a POM-Tactic, the information about the tactic will appear in the preferences for the POM-Tactic and the users of the platform. However, this tactic will not be a default POM-Tactic and the users must change the preference to enable this tactic.
